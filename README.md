# SLIDER - RETO NUWE

> Una app que permite visualizar los comentarios uno en uno en forma de slider.

## Installation

    Paso 1: instalar las dependencias del proyecto.

```shell
    # instalar las dependencias del proyecto
    npm i  o npm/yarn i
```

    paso 2: Inicializar el servidor.

```shell
    # iniciar el servidor
    npm start
    npm run dev ...
```
